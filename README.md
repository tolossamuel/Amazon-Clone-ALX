## Project Overview

This project is a fully functional Amazon Clone built as part of the final software engineering projects for the ALX Program. It showcases a comprehensive e-commerce platform with key features similar to the original Amazon website.

## Technologies Used

* **ReactJs:** A JavaScript library for building user interfaces.
* **Firebase:** Used for hosting, authentication, and storage.
  * **Firebase Hosting:** For deploying the web application.
  * **Firebase Auth:** For user authentication and secure login/logout functionality.
  * **Firebase Storage:** To store user data securely.
* **React-Dom:** For managing the DOM elements in the app.
* **React Redux:** For state management across the application.
* **Stripe:** For processing payments securely.

## Features

* **User Authentication:** Secure login and logout functionalities.
* **Shopping Cart:** Add items to the cart, view cart details, and remove items.
* **Payment Processing:** Seamless integration with Stripe for handling payments.
* **Clean and Intuitive UI:** A user-friendly interface for an enhanced shopping experience.
